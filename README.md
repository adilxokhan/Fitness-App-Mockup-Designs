# Fitness App UI/UX Mockups

Welcome to my fitness app design project! This repository contains the UI/UX mockups for a modern, user-centric fitness tracking application designed for both desktop and mobile experiences. The designs are inspired from "Myfitnesspal" application , The present MyFitnessPal interface makes poor use of white space, lacks a consistent colour palette, and neglects to draw attention to crucial information on the home page. Users find it challenging to promptly access essential functions and maintain motivation to monitor their health progress due to these design flaws.During the evaluation of the existing MyFitnessPal app interface, several usability and visual design issues were identified. These errors affect both user engagement and the overall user experience. The following are the main UI problems found:
• Inconsistent color scheme
• Poor use of white space
• Cluttered Pages
• Lack of visual hierarchy
• Lack of iconography
• Important information not prioritized.
Based on the identified UI issues, an improved and modernized version of MyFitnessPal was developed. The redesigned interface focuses on simplicity, consistency, and better visibility of important information to enhance user experience and engagement.
Three main dashboards were deisgned for executives for regular and efficient trackinng of user statuses that will later help in developing strategies.


## 📱 About the Project
This app is designed to help users track their health goals, log workouts, and view progress analytics with ease. The design focuses on clean lines, high contrast for readability, and an intuitive layout.

## 🎨 Design Features
- **Responsive Layouts:** Designed with both **Desktop (PC)** and **Mobile** viewports in mind.
- **Multiple Pages:** Includes views for:
    - User Dashboard
    - Workout Tracking
    - User Profile
    - Progress Charts
