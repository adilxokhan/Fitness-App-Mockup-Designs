# Fitness App UI/UX Mockups

Welcome to my fitness app design project! This repository contains the UI/UX mockups for a modern, user-centric fitness tracking application designed for both desktop and mobile experiences.

## 📱 About the Project
This app is designed to help users track their health goals, log workouts, and view progress analytics with ease. The design focuses on clean lines, high contrast for readability, and an intuitive layout.

## 🎨 Design Features
- **Responsive Layouts:** Designed with both **Desktop (PC)** and **Mobile** viewports in mind.
- **Multiple Pages:** Includes views for:
    - User Dashboard
    - Workout Tracking
    - User Profile
    - Progress Charts
## 🛠 Tools Used
Figma is mainly used for designs of containers , however icons and other images were sources from google anbd pinterest.
